# ✦ MANIFEST – Your Daily Abundance Tool

A private, all-in-one dashboard for goals, affirmations, gratitude & vision board.

**Live demo:** [https://venkymayara.github.io/manifest-dashboard/](https://venkymayara.github.io/manifest-dashboard/)

---

## ✨ Features

- 🎯 **Goals** – Track progress by category (Career, Health, Wealth, Love, Growth, Travel)
- 🗣 **Affirmations** – Daily affirmations + add your own custom ones
- 🙏 **Gratitude Journal** – Daily entries with streak tracking
- 🖼 **Vision Board** – Upload images or use default cards
- 📊 **Analytics** – Built-in admin panel + Google Analytics support

---

## 🚀 Quick Start (for developers)

1. Clone or download `index.html`
2. Open in any browser – no server needed
3. All data stays in your browser's localStorage
4. To enable multi-user analytics, add your Google Analytics ID (see code)
